# StanleySync App Tester Handoff

This handoff is for hosted web-demo testers.

Send testers only the hosted demo link, login information, and feedback instructions.

## What StanleySync Does

StanleySync App helps service businesses:

- Capture quote requests
- Review quotes
- Convert quotes into jobs
- Track work
- Create invoices
- Export customer-ready PDFs

## Tester Instructions

1. Open the hosted demo link.
2. Log in with the account provided.
3. Open the dashboard.
4. Create or review a quote.
5. Convert the quote to a job.
6. Add a job note.
7. Create an invoice.
8. Export PDFs.
9. Send feedback.
