# StanleySync App Beta Tester Guide

Use the hosted StanleySync App demo link provided by the StanleySync owner.

## Start

1. Open the demo link.
2. Log in with the provided account.
3. Follow the workflow on the dashboard.

## Recommended Workflow

1. Create a quote.
2. Review the quote.
3. Convert the quote to a job.
4. Add a job note.
5. Create an invoice.
6. Export PDFs.
7. Submit feedback.

## Feedback Questions

- Was it clear what to do first?
- Was quote intake easy?
- Was job tracking useful?
- Did the invoice and PDFs look professional?
- What confused you?
- Would this help your business?
- What would make you pay for it?

## Notes

StanleySync Labs is a future pro module and is not part of the general-service beta workflow.
