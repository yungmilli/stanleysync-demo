# Send To Testers

Send testers only the hosted web demo link, login information, and feedback instructions.

## Include

- Demo link
- Login email
- Temporary password or separate password message
- What to test
- Feedback link or feedback questions

## Do Not Include

- Internal owner notes
- Developer folders
- Technical setup material
- Production secrets

## Short Message

Hi [Name],

Here is the StanleySync App beta demo:

[PASTE DEMO LINK]

Login:

[PASTE EMAIL]

Password:

[SEND PASSWORD SEPARATELY]

Please test:

1. Create or review a quote.
2. Convert the quote to a job.
3. Add a job note.
4. Create an invoice.
5. Export PDFs.
6. Send feedback.

Feedback link:

[PASTE FEEDBACK LINK]
