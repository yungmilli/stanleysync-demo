# StanleySync App Tester Guide

StanleySync App beta testing is web-demo first.

Testers should receive only the hosted demo link, login information, and feedback instructions.

## What Testers Need

- Hosted demo link
- Login email
- Password
- Feedback instructions

## Test Workflow

1. Open the hosted demo link.
2. Log in.
3. Create or review a quote.
4. Convert the quote to a job.
5. Add a job note.
6. Create an invoice.
7. Export PDFs.
8. Send feedback.

## Feedback

Ask testers:

- Was it easy to log in?
- Was quote intake clear?
- Was job tracking useful?
- Did invoices and PDFs look professional?
- What confused you?
- Would this help your business?
- What would make you pay for it?

## Internal Setup

Developer and local package notes are kept in `docs/internal/` and should not be sent to customers.
