# StanleySync App Beta Tester Start Here

StanleySync App helps service businesses manage quotes, jobs, invoices, and customer-ready PDFs.

## What You Need

- Demo link from the StanleySync owner
- Login email
- Login password

Use the hosted demo link only. A `localhost` link works only on the owner's computer and will not work for testers.

## Start Testing

1. Open the demo link.
2. Log in with the account provided.
3. Open the dashboard.
4. Create or review a quote.
5. Convert the quote to a job.
6. Update the job status and add a note.
7. Create an invoice.
8. Export the quote, work order, and invoice PDFs.
9. Submit feedback.

## Suggested Test Path

Quote -> Review -> Convert to Job -> Add Job Note -> Create Invoice -> Export PDF -> Send Feedback

## What To Look For

- Is quote intake clear?
- Is job tracking easy to understand?
- Do invoices and PDFs look professional?
- What feels confusing?
- What would make this useful enough to pay for?

## Notes

StanleySync Labs and calibration-specific tools are not part of the general beta workflow unless the StanleySync owner specifically asks you to test them.
